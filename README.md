# OQUIC
Introduction
====

OQUIC is a security gateway originated by [OPPO](https://zh.wikipedia.org/wiki/OPPO).
It is based on [Nginx-quic](https://hg.nginx.org/nginx-quic/) and will develop other features to deploy security gateways.
